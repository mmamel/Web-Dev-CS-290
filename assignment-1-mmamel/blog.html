Assignment 1 Blog Post - 9/28/2019

My biggest challenge completing this assignment was understanding how
Git worked. The only experience I had with Git/GitHub was just to
upload code I had done so others could view it online. It took me some
time to get used to the proccess of git add into git commit into git push.
I had the most trouble with git pull because I have two devices that I code
on so I ran into some merging bugs.

Because I had completed some free online html courses prior to the course,
I was familiar with the basic tags and layout of html code. The only resources
that I used was the W3schools website to remember the exact names of the tags
and the exact syntax, like how to write the img tag.

Assignment 2 Blog Post - 10/19/2019

I had a lot of trouble with positioning the boxes and flexboxes. The
most problematic issue was when I tried to line up the left border column
containing the update button, with the results section. I solved this by
making the boxes inline and adjusting the padding accordingly.

I learned how to manage the layout of elements in a responsive manner using
the position property and flex boxes. I already knew about styling text but
what I didn't know was that the button could be styled into a circle and expanding
the plus sign

The resources that I used were the class notes as well as W3School. These provided
all the documentation I needed to be successful

To complete this assignment, I had to google how to manage the size of the container
so that I could make all the images uniform
